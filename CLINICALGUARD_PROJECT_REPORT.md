# CLINICALGUARD: PROJECT PERFORMANCE REPORT (V2.0)

## 1. Executive Summary
ClinicalGuard is an advanced AI-powered SaaS platform designed for high-stakes Clinical Trial Data Integrity. By combining Decision Tree Machine Learning with Gemini 2.5 Flash reasoning and SHA-256 blockchain anchoring, it provides a bulletproof audit trail for medical research.

## 2. Technical Architecture
- **AI Core**: Python-based Decision Tree model trained on curated clinical datasets.
- **Reasoning Engine**: Google Gemini 2.5 Flash (Contextual API).
- **Security**: SHA-256 Data Fingerprinting + JWT RBAC.
- **Database**: MongoDB (Forensic Logging + Record Persistence).
- **Frontend**: React V2 (Vite + Framer Motion + ApexCharts).

## 3. Forensic Pipeline (The 9 Steps)
1. **Input Sanitization**: Multi-type validation for all 13 features.
2. **Contextual Enrichment**: Mapping Age Groups and calculated Risk Scores.
3. **SFO Blind Spot Scan**: Immediate flagging of manual overrides (zero-risk fraud).
4. **ML Inference**: Binary classification (AUTHENTIC vs. MANIPULATED).
5. **Confidence Scoring**: Probability distribution analysis.
6. **Blockchain Anchoring**: Generating SHA-256 signature.
7. **Gemini Reasoning**: Human-readable explanation of forensic indicators.
8. **Audit Persistence**: Immutable record storage in MongoDB.
9. **Dispatch**: Encrypted JSON response to frontend.

## 4. Key Implementation Decisions
- **Glassmorphism UI**: High-end medical SaaS aesthetic for trust and clarity.
- **ApexCharts Integration**: Real-time animated data visualization.
- **Mentor AI**: Seamless chatbot integration using a custom React ref-triggered event system.

## 5. Security Protocols
- **JWT Storage**: Session-only storage to prevent XSS persistence.
- **Role Decoupling**: Frontend routes guarded by decoded JWT roles (Admin, Monitor, Investigator).
- **Data Hashing**: Cryptographic identity for every single trial entry.
